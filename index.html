Dữ liệu & Backup

Export để phân tích · Import để restore

📤 Export toàn bộ dữ liệu

Tải xuống file JSON để backup hoặc import vào thiết bị khác. CSV để mở bằng Excel / Google Sheets.

📋 Export báo cáo insight

File text với toàn bộ insight — dễ paste vào Claude hoặc Notion để phân tích tổng quan sau mỗi tháng.

📥 Import / Restore

Chọn file JSON đã export. Dữ liệu sẽ được gộp với dữ liệu hiện có — không ghi đè, không trùng lặp.

📊 Tóm tắt số liệu

Số liệu tổng hợp dạng text — copy paste vào Claude để phân tích pattern đọc sách sau một tháng.
